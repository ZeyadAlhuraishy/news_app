# 📰 News App

A modern mobile news application built using Flutter and Dart that delivers the latest news through a clean and user-friendly interface.

## 🚀 About the App

This application allows users to browse and read news from different categories in an organized and responsive layout.  
The app focuses on providing a smooth reading experience with fast performance and modern UI design.

## ✨ Features

- Browse news by categories
- Read latest headlines
- Responsive and modern UI
- Smooth navigation experience
- Fast and lightweight performance
- Clean news layout
- API integration for real-time news

## 🛠️ Technologies Used

- Flutter
- Dart
- REST API
- Responsive UI Design
- Git & GitHub

## 📱 App Design

The application was designed with a clean and minimal UI/UX approach to provide:
- Easy navigation
- Comfortable reading experience
- Organized content display
- Responsive layouts

## 🎯 Project Goal

The goal of this project is to practice API integration, state management concepts, and responsive mobile app development using Flutter.
